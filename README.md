# fortune

fortune command in Go.

## Usage

```sh
$ ./fortune -h
Usage of ./fortune:
  -m string
        Print out all fortunes which match the basic regular expression pattern.
```

## Installation

```sh
go install github.com/nabetama/fortune@latest
```
